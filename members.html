Project and

Here are a list of most important projects I have done. You can also visit my github to see the complete list

Android TV launcher and TicTacToeGame Android App
Github link
This is the TV Launcher I modified with Skyworth Software team in 2020. It is developed in Java using Android TV. We modified the UI content and FocusView from the original Google Leanback Library. Added some movie and media information by third party application. Also Added support for specific control needs for TV users. I also added an Android app project. In this project, I used Android Studio to develop a 2-player-TicTacToe Game with great UI.
JumpAhead Website
Github link Demo
JumpAhead currently runs in-person jump rope and interactive healthy snack activities at local elementary schools. Used Django and Heroku to create an online platform to deliver the content of the Jump ahead after-school program. Developed a visual progress tracking feature that corresponds to the number of tasks checked off in each module. Added card payment feature for users to purchase subscription.
Virtual Stock Stimulator
Github link Demo
Developed a virtual market stock stimulator website using React.js as our frontend framework, and Express, Node as the backend framework. Stored user's information in MongoDB and added third-party market news API. Styled the frontend UI using Bootstrap and deployed on Heroku. Now is used by Reynolds Fund, a business club in UNC.
